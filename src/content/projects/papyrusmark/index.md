## Overview

Papyrusmark is a simple, elegant Markdown editor for the desktop. It focuses on speed and a distraction-free writing experience, with a live preview alongside the editor, light and dark themes, keyboard shortcuts and one-click export. It runs natively on Windows, macOS and Linux.

## Features

- **Live preview** — see the rendered Markdown next to what you type.
- **Light & dark themes**, plus keyboard shortcuts for a faster workflow.
- **Easy export** of your documents.
- **Privacy-first** — everything is stored locally, no cloud syncing or tracking.

## Stack

- **Vue 3 + Vuetify** for the interface (Material Design components).
- **Pinia** for state management.
- **Tauri** (Rust) for a lightweight, native cross-platform shell — much smaller and lighter than an Electron equivalent.
- **marked** for Markdown rendering.
