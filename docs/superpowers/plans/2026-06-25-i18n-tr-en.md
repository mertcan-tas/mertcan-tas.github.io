# TR/EN Çok Dilli Site — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Astro sitesine TR/EN çok dilli destek eklemek — 3 makalenin Türkçe çevirileri, tüm arayüzün iki dilde sunulması ve header'da TR/EN dil değiştirici.

**Architecture:** Astro yerleşik i18n (`defaultLocale: en`, `prefixDefaultLocale: false`) → EN kökte, TR `/tr/` altında. Aktif dil `Astro.currentLocale` ile okunur. İçerik tek `blog` koleksiyonunda `lang` + `translationKey` alanlarıyla; TR makaleler aynı klasörde `index.tr.md` olarak. Statik arayüz metinleri `src/i18n/ui.ts` sözlüğünde.

**Tech Stack:** Astro 5, @astrojs/sitemap, @astrojs/rss, Tailwind, TypeScript, pnpm.

## Global Constraints

- Paket yöneticisi: **pnpm** (`pnpm run build` = `astro check && astro build`).
- Doğrulama birim test değil **build temelli**: `pnpm run build` hatasız geçmeli (`astro check` tip hatası vermemeli) ve beklenen rotalar `dist/` altında üretilmeli. Proje bir test runner içermiyor; eklenmeyecek (YAGNI).

- Commit mesajlarına **Co-Authored-By Claude satırı eklenmeyecek** (kullanıcı tercihi).
- Tag'ler iki dilde de İngilizce/teknik kalır.
- Türkçe slug'lar (onaylı): `1`→`django-rest-framework-swagger-dokumantasyonu`, `2`→`django-ortam-degiskenleri-python-decouple`, `3`→`django-redis-rq-ile-arka-plan-isleri`.

- Menü: Posts→Yazılar, Projects→Projeler, Tags→Etiketler.

- [ ] **Step 5: URL yardımcıları — `src/i18n/utils.ts`**

```ts
import { defaultLang } from './ui';

export type Lang = 'en' | 'tr';

export function getLangFromUrl(url: URL): Lang {
  const [, seg] = url.pathname.split('/');
  if (seg === 'tr') return 'tr';
  return 'en';
}

/** "/tr/posts/" -> "/posts/" ; "/tr" -> "/" ; "/posts/" -> "/posts/" */
export function stripLocale(pathname: string): string {
  const p = pathname.replace(/^\/tr(?=\/|$)/, '');
  return p === '' ? '/' : p;
}

/** logical path + target lang -> concrete path */
export function withLocale(pathname: string, lang: Lang): string {
  const logical = stripLocale(pathname);
  if (lang === defaultLang) return logical;
  return logical === '/' ? '/tr/' : '/tr' + logical;
}
```

- [ ] **Step 6: İçerik yardımcıları — `src/i18n/content.ts`**

```ts
import { getCollection } from 'astro:content';
import type { Lang } from './utils';

export async function getPostsByLang(lang: Lang) {
  const all = await getCollection('blog');
  return all.filter((p) => (p.data.lang ?? 'en') === lang);
}

export async function getPostByTranslationKey(key: string | undefined, lang: Lang) {
  if (!key) return undefined;
  const all = await getCollection('blog');
  return all.find((p) => p.data.translationKey === key && (p.data.lang ?? 'en') === lang);
}
```
